# Notion Skills

Agent skills for working with [Notion](https://notion.so).

## Install

We recommend the [skills CLI](https://github.com/vercel-labs/skills).

```bash
# Install all skills
npx skills add makenotion/skills

# Install a specific skill
npx skills add makenotion/skills --skill notion-cli
```

## Available Skills

| Skill | Description |
| ----- | ----------- |
| `notion-cli` | Install and use the Notion CLI (`ntn`) for workers, public API requests, and file uploads. |
